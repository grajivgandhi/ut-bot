## Troubleshooting:

# Check if bot is running
heroku ps

# View last 100 log lines
heroku logs --num=100

# Filter errors
heroku logs --grep ERROR

# Restart bot
heroku restart

# Update code and redeploy
git add .
git commit -m "Update message"
git push heroku main
