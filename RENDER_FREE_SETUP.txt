## STEP 4: Create Render Account (1 minute)
# Go to: https://render.com
# Click "Sign up"

# Click "Authorize render"
# Done!

## STEP 5: Deploy Your Bot (2 minutes)
# Log into https://render.com/dashboard
# Click "New +" button (top right)
# Select "Web Service"
# 
# Find your ut-bot repository and click "Connect"
# 
# Fill in the form:
#   Name: ut-bot-trading
#   Root Directory: (leave blank)
#   Runtime: Python 3
#   Build Command: pip install -r requirements.txt
#   Start Command: python main.py
#
# Click "Advanced" to expand options
# 
# Scroll down to "Environment"
# Add these variables:
#   
#   API_KEY = your_actual_binance_api_key
#   API_SECRET = your_actual_binance_api_secret
#
# IMPORTANT: Use your NEW API keys, not the old exposed ones!
#
# Click "Create Web Service"
# 
# Wait 2-3 minutes for deployment
# You'll see: "Your service is live!"

## STEP 6: Monitor Your Bot

## Troubleshooting

Bot won't start?
1. Check API_KEY and API_SECRET in Render environment
2. Check logs for error messages
3. Verify API keys are correct on Binance
4. Click "Restart Service" in Render

Bot is slow?
- Normal on free tier (shared CPU)
- Trading bot doesn't need high speed
- Processes one candle per interval

Bot disappeared?
- Render may restart monthly (brief)
- It auto-restarts
- Check Render dashboard for status
